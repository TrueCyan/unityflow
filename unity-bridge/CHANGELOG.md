## [0.5.0] - 2026-02-03

### Added
- Play Mode control API (play/stop/pause/step)
- Scene load API with Build Settings scene list
- Scene View camera control (pivot, rotation, size, orthographic)
- Frame object focus functionality
- HTTP POST method support

### Added
- Initial release
- HTTP server with main thread dispatch
- Screenshot capture (Scene/Game View)
- Prefab preview rendering
- Animation multi-frame capture
- Animator runtime state query
- Runtime hierarchy tree
- Inspector component data
- Console log capture
- Editor state query
- EditorWindow UI with connection log and quick test
